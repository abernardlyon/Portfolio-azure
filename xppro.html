Expérience Professionnelle : Stage en Développement Web

Du 12 mai au 4 juillet 2025, j'ai réalisé un stage au sein de l'entreprise Limoog. C'est une structure spécialisée dans la conception de logiciels de planning et d'applications web pour des associations, laboratoires et entreprises.

Ce stage m'a offert un environnement idéal pour mettre en pratique mes connaissances théoriques, notamment en développement back-end (PHP), manipulation de bases de données et intégration d'API complexes.

1. Automatisation de fichiers Excel (Projet ELIA)

Technologies : PHP, Bibliothèque PHPExcel
Contexte : Le laboratoire ELIA avait besoin de générer automatiquement des fichiers Excel de synthèse contenant les résultats d'analyses d'échantillons.

J'ai développé un script PHP permettant de :

  • Récupérer les informations issues de la base de données et des répertoires.
  • Générer un fichier .xlsx comprenant plusieurs onglets formatés (comptages et saisies de genres).
  • Rendre le fichier directement téléchargeable et exploitable par les opérateurs pour faciliter la relecture humaine.

2. Développement et Intégration sur WordPress

Technologies : PHP, API JSON, CMS WordPress
Contexte : Automatisation de l'affichage et de la publication de contenus pour un client (ASVETTE).

Mes réalisations sur WordPress se sont divisées en deux parties :

  • Création de Plugin : Développement d'un plugin autonome avec shortcode pour afficher dynamiquement une liste de sorties sportives depuis une API externe, avec plusieurs styles d'affichage (cartes, listes).
  • Publication Automatisée : Création d'un script PHP sécurisé par token permettant de générer automatiquement des articles WordPress (titre, contenu, image mise en avant, catégorie) à la réception d'une requête JSON.

3. Intégration de l'API Graph Meta (Instagram & Facebook)

Technologies : PHP, cURL, requêtes HTTP, Meta Graph API
Contexte : Automatisation de la publication de contenus visuels sur les réseaux sociaux sans passer par l'interface manuelle.

J'ai conçu un programme capable de :

  • Gérer l'authentification et l'échange de tokens de sécurité (token court vers token long).
  • Publier automatiquement des images simples et des carrousels sur un compte Instagram Business.
  • Publier des posts sur une page Facebook liée.

Environnement Technique Global

Durant cette expérience, j'ai évolué avec les outils suivants : Visual Studio Code, PHP (Back-end), HTML/CSS (Front-end), MySQL, GitLab, WordPress et diverses API REST.
